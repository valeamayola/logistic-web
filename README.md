<h1 align="center"> Logistic </h1>

## Descripción del proyecto 

Logistic es un proyecto personal creado para la practica de tecnologías y herramientas, es una simulación de página web de una empresa de logística y distribución. Cabe aclarar que la empresa es inexistente y los datos para la construcción de la misma fueron inventados.

## Comenzando 🚀

Primero, ejecute el servidor de desarrollo:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Abra [http://localhost:3000](http://localhost:3000) en su navegador para ver el resultado.

## Tecnologías utilizadas 🛠️

* [Next.js](https://nextjs.org/docs) 
* [TypeScript](https://www.typescriptlang.org/) 
* [React](https://es.react.dev/)
* [TailwindCSS](https://tailwindcss.com/)
